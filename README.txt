# 💰 Smart Budget Tracker

A simple and stylish web app to help you track your income and expenses.

> Created with HTML, CSS (modern animations), and JavaScript (localStorage, dynamic DOM, sound effects, and filters).

## ✨ Features

- 📈 Real-time balance updates
- ➕ Add income or expenses with categories
- 🎯 Filter by category (Salary, Food, Transport, Fun, Other)
- 💾 Auto-save transactions in localStorage
- 🔊 Sound effects when adding or removing items
- 🧼 Clean & modern UI with subtle animations
